# 💪 健身记录

多人共享的健身记录网站。

## 功能

- 🏋️ 力量训练记录（动作、重量、组数、次数）
- 🏃 有氧训练记录（项目、时长、距离、卡路里）
- 📊 训练趋势图表（近7天/30天）
- 💪 肌肉部位分布统计
- 👥 训练广场（查看所有人的训练动态）
- 📤 数据导出/导入（JSON 备份）
- 🔐 注册登录，数据隔离

## 部署

```bash
npm install
npm start
```

打开 http://localhost:3000

## 技术栈

- 后端：Node.js + Express
- 前端：原生 HTML/CSS/JS
- 数据存储：JSON 文件
